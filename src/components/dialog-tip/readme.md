## dialog-tip

### 使用指南

> 消极提示

```html
dialog-tip(
    v-model="showDialog"
    tip="您还未被授权登录<br/>请联系您的管理员给您添")
```

> 积极提示，自定义事件

```html
dialog-tip(
    v-model="showDialog"
    type="success"
    tip="您将通过虚拟号转接至客户，<br/> 成功接通后可查看该客户的详细信息"
    btn="立即呼叫"
    :btnCb="btnCb")
```
