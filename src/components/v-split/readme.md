## v-split

### 使用指南

> 纯文字分割

```html
<v-split txt="你到达了内容的尽头"></v-split>
```

> 带图片分割

```html
<v-split :icon="emptyIcon" iconTxt="无任何相关结果<br/>请检查一下申请编码是否正确～"></v-split>
```

```
data() {
    return {
        emptyIcon: require('./img/icon-no_result.png')
    }
}
```
